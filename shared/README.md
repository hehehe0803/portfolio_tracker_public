# Shared Contracts

Cross-service schemas, DTOs, and utilities for Python and TypeScript consumers.

Canonical vocabulary currently lives in `shared/python/contracts.py` and `shared/typescript/contracts.ts` and covers:

- `AssetSnapshot`
- `TransactionRecord`
- `ImportArtifactContract`
- `AlertRuleContract`
- `AlertEventContract`
- `TagContract`
- `NoteContract`
- `IngestionEvent`

Packaging and versioning workflows will be defined in shared-domain tasks.
