# Design System: Portfolio Tracker / Aegis Terminal

## 1. Visual Theme & Atmosphere
A dark fintech command console inspired by Stitch's Aegis Terminal direction. It should feel like a personal treasury cockpit: precise, calm, information-dense, and premium without looking noisy or cosplay-terminal.

## 2. Color Palette & Roles
- Primary signal green: `#9BF56A`
- Secondary amber: `#F4B84E`
- Tertiary cyan: `#73E6FF`
- Canvas: `#090D10`
- Surface: `#10161A`
- Elevated surface: `#151D22`
- Quiet text: `#94A39A`
- Bright text: `#F3F8F4`
- Destructive: `#FF6B6B`

## 3. Typography
- Use Geist Sans for interface text and Geist Mono for timestamps, quantities, and operational metadata.
- Big KPI numbers should feel polished and premium, not retro-terminal chunky.
- Labels are small uppercase with wide tracking.

## 4. Component Styling
- Panels are rounded, dark, softly elevated, and lightly bordered.
- Buttons are compact and intentional: filled green primary, subtle dark ghost secondary.
- Badges are semantic chips for brokers, asset classes, and states.
- Charts use green as the main portfolio line, with amber/cyan reserved for supporting distinctions.

## 5. Layout Principles
- Prefer one hero shell with summary + controls.
- Use clear modules: KPIs, performance chart, allocation, holdings, activity.
- Keep dense information but maintain breathing room with rounded sub-panels.

## 6. Motion
- Use subtle pulse and cursor animations only.
- No excessive glow or constant movement.

## 7. Anti-patterns
- Avoid generic green-on-black hacker cosplay.
- Avoid pure black and razor-sharp square boxes everywhere.
- Avoid giant empty areas without telemetry or purpose.
- Avoid multi-accent chaos; green/amber/cyan each need semantic jobs.
